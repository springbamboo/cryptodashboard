# 開発手順

## 編集者がやること

- GitHub
  - issueを作る
- VSCode
  - `issue1`ブランチを作る (名前はissueの番号に合わせる)
  - ブランチに入る (checkout)
  - 編集する!💪
  - プラスボタンでファイルをcommit対象を決める。 (stage)
  - commitメッセージを書いて、チェックボタン。 (commit)
  - GitHubにプッシュする (Push)
- Github
  - "Compare & Pull Request"を押す
  - 本文に`fix #1` (issueの番号に合わせる)

## レビュワーがやること

- Github
  - Pull Requestを開く
  - 変更を確認する。(Code review)
  - "いいね"のコメントをする
  - "Review & Merge"を押す
  - Delte Branch (任意)
